    [전자계산기] 보조 기억 장치

    2 분 소요

    INDEX

    • 보조 기억 장치
      • 자기 디스크(Magnetic Disk)
      • 자기 테이프(Magnetic Tape)
      • 광 디스크(OD, Optical Disk)
        • 블루레이 디스크(Blu-ray)






    보조 기억 장치

    데이터나 파일을 보관하는 비휘발성 메모리로 용량이 크고 액세스 속도가 느린 기억 장소

    자기 디스크, 자기 테이프, 자기 드럼, DVD, CD-ROM, 플로피 디스켓 등이 있다

    • 전원이 차단되어도 기억 내용이 소멸되지 않는다
    • 액세스 속도는 느리지만 대용량 장치
    • 가격이 저렴하고 많은 자료를 영구적으로 보관할 수 있다

    자기 디스크(Magnetic Disk)

    직접 저장 장치(DASD, Direct Access Storage Device)로서 순차 처리와 랜덤 처리를 병행할 수 있는 기록 매체

    • 액세스 시간이 빠르다
    • 레코드의 추가/삭제/정정하기 쉽다

    등선 속도(CLV, Constant Linear Velocity)
    디스크 전송 속도가 처음부터 마지막까지 일정한 방식으로 디스크의 안쪽에 저장할 때는 회전 속도가 빠르고 바깥쪽에 저장할 때는 회전 속도가 느리게 되어 소음이 적으나 회전 구동 장치가 복잡

    등각 속도(CAV, Constant Angular Velocity)
    디스크 회전 속도를 일정하게 하여 디스크의 회전각에 따라 데이터를 저장하는 방식으로 디스크 내부 면과 외부 면의 회전 속도 차이로 외부 면의 저장 밀도가 떨어져 공간 낭비를 가져올 수 있으나 접근 속도가 빠른 방식

    구조 내용
    자기 원판(Magnetic Circle Board) 자성 재료가 입혀진 원판을 하나의 축으로 연결시켜 놓은 기억 기구
    액세스 암(Access Arm) 기억 매체 사이를 수평으로 움직이는 기계 장치
    읽기/쓰기 헤드(Read/Write Head) 액세스 암의 앞에 위치하여 데이터를 기록하고 읽어들이는 장치
    트랙(Track) 회전축을 중심으로 이루어진 동심원
    섹터(Sector) 트랙을 일정한 구역으로 나누어 쓰기 쉽게 만들어 놓은 것
    실린더(Cylinder) 여러 장의 디스크 팩에서 같은 지름을 갖는 트랙의 집합. 참고로, 실린더의 개수와 트랙의 개수는 같다


    • 자기 디스크의 접근 시간
      • 탐색 시간(Seek Time)
        헤드가 데이터를 읽기 위해서 해당 트랙까지 도달하는데 걸리는 시간
      • 회전 대기 시간(Rotational Delay Time, Latency Time)
        디스크 장치가 회전하여 해당 섹터가 헤드에 도달하는데 걸리는 시간
      • 전송 시간(Transfer Time)
        데이터가 전달되는데 걸리는 시간
      • 접근 시간(Access Time)
        읽기 신호나 쓰기 신호를 접하고 나서 데이터가 읽히거나 쓰여지는데 걸리는 시간을 의미

        Seek Time + Rotational Delay Time + Transfer Time

    Staging : 자기테이프나 디스크에 있는 블록들을 주기억 장치에 미리 옮겨 놓는 기능

    Spooling : 처리할 데이터를 디스크나 테이프에 잠시 저장시켰다가 나중에 다른 장치가 이용하게 하는 기능

    • 디스크의 RAID의 기술 포스트 이동하기

    자기 테이프(Magnetic Tape)

    순차적으로 데이터를 기록하고 읽는 방식으로 가격이 저렴하고 보관에도 어려움이 없을 뿐 아니라 대용량의 기억 장치이기 때문에 대량의 데이터를 보관하는데 많이 사용된다

    • 일괄 처리 시 경제적이기 때문에 백업용으로 많이 사용
    • 순차 처리만 가능한 SASD(Sequential Access Storage Device)
    • 기록 밀도가 좋다
    • 디스크에 비해 대량의 자료를 장시간 보관하기 유리
    • 블록 단위로 데이터를 전송
      • 블록 : 테이프에서 입출력하기 위하여 여러 개의 논리 레코드를 묶어 놓은 개념으로 물리적 레코드라고도 한다
      • 블록과 블록 사이의 Gap을 IBG(Inter Block Gap)라 한다. IBG에는 아무것도 기억되지 않는다
      • B/F(Blocking Factor) : 하나의 블록에 들어 있는 논리 레코드의 개수
    • 주소 개념이 없고 7트랙(BCD), 8트랙(ASCII), 9트랙(EBCDIC) 용이 있다
    • 세로로 하나의 문자를 기록하고 9트랙용이라면 하나의 트랙은 오류 체크를 위한 패리티 비트 트랙으로 사용되고 나머지 8개의 트랙에 EBCDIC 코드로 문자 기록을 위한 각 비트가 기록된다

    광 디스크(OD, Optical Disk)

    빛의 반사를 이용하여 자료를 읽어내는 저장 매체

    블루레이 디스크(Blu-ray)

    고선명(HD) 비디오를 위한 디지털 데이터를 저장할 수 있도록 소니가 주도하는 BDA(블루레이 디스크 협회)에서 정한 광 기록 방식 저장매체

    • DVD보다 데이터 전송 속도가 4~5배 빠른 차세대 대용량 광디스크
    • 비디오 포맷은 DVD와 동일한 MPEG-2 기반 코덱이 사용
    • 단층 기록면을 가지고 12cm 직경에 25GB의 데이터를 저장할 수 있다
    • 기술 규격으로 BD-ROM(읽기 전용), BD-R(기록 가능), BD-RE(재기록 가능)가 있다
