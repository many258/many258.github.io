    [전자계산기] 레지스터(Register)

    2 분 소요

    INDEX

    • 레지스터(Register)
      • 특수 전담 레지스터
      • 데이터 레지스터(DR, Data Register)
      • 주소 레지스터(AR, Address Register)
      • 상태 레지스터(Status Register)






    레지스터(Register)

    주기억 장치보다 작은 개수의 주소 비트를 가지고 기억된 정보를 신속하게 접근할 수 있는 기억 장소로 중앙 처리 장치 안에 임시 기억 장치

    중앙 처리 장치의 주소 선은 주소 레지스터(AR)와 연결되어 있고 데이터 선은 데이터 레지스터(DR)와 연결되어 있다. 따라서 메모리의 용량은 AR과 관계가 있고 한 번에 엑세스되는 워드 크기는 DR과 관계가 있다

    특수 전담 레지스터

    • 프로그램 카운터(PC, Program Counter)
      • 로케이션 카운터(LC, Location Counter)라고도 한다
      • 다음 명령어의 주소 값을 기억하고 있는 레지스터
      • 명령어의 수행 순서를 나타내는 레지스터
    • 메모리 번지 레지스터(MAR, Memory Address Register)
      • 메모리에 접근하고자 할 때 접근하고자 하는 곳의 주소 값을 기억하고 있는 레지스터
      • 기억 장치의 용량과 관계가 있는 레지스터
    • 기억 레지스터(MBR, Memory Buffer Register)
      • 주기억 장치에서 정보를 읽을 때 읽은 정보를 기억시키거나 주기억 장치에 정보를 기억시키는 레지스터
      • 기억 장치의 WORD 크기와 관계가 있다
    • 명령 레지스터(IR, Instruction Register)
      • 현재 수행하고 있는 명령어를 기억하는 레지스터
      • 명령 레지스터에 있는 정보는 연산에 사용되지 않고 명령 수행을 제어하기 위해 사용

    데이터 레지스터(DR, Data Register)

    명령어 수행 시 수행에 필요한 데이터를 기억시키는 레지스터로 기억 장치의 워드 크기와 관계가 있다

    • 누산기(ACC, Accumulator)
      • 연산기능 수행 시 중심이 되는 데이터 레지스터로서 연산 장치에 연결되어 있다
      • 연산시 연산에 사용되는 데이터 및 연산 결과를 임시적으로 기억하는 레지스터
    • 기억 레지스터(MBR, Memory Buffer Register)
      • 주기억 장치에서 읽은 정보나 저장시킬 정보를 임시적으로 기억시키는 레지스터
    • 범용 레지스터(GPR, General Purpose Register)
      • 함수 연산 기능을 수행하는 명령어에서 사용할 수 있도록 설계된 레지스터

    주소 레지스터(AR, Address Register)

    주기억 장치에 접근하기 위한 주소의 일부 또는 전부를 기억시키는 레지스터로 기억 장치 용량과 관계가 있는 레지스터

    • 프로그램 카운트(PC, Program Counter) 다음 명령어의 주소 값을 기억하고 있는 레지스터

    • 메모리 번지 레지스터(MAR, Memory Address Register) 메모리에 접근하고자 할 때 접근하고자 하는 곳의 주소 값을 기억하고 있는 레지스터

    • 인덱스 레지스터(Index Register)
      • 주소 변환 및 계산에 필요한 수를 기억
      • 주소 수정에 필요한 주소값을 기억
      • 배열의 시작 주소와 같은 반복 계수를 기억시키는 용도로 사용
    • 베이스 레지스터(Base Register)
      • 하나의 서브 프로그램 루틴 또는 데이터의 블록에 대하여 기준이 되는 주소를 기억시키는 레지스터
      • 프로그램을 재배치하는 경우 상대적인 변위 값을 기억시키는데 사용
      • 명령이 시작되는 최초의 번지를 기억

    상태 레지스터(Status Register)

    프로그램 제어를 위한 상태 정보를 기억시키는데 필요한 레지스터를 의미

    • 플래그 레지스터(Flag Register)
      • = 프로그램 상태 워드 레지스터(PSWR, Program Status Word Register)
      • 프로그램의 수행이 진행되고 있을 때 매 순간마다 프로그램의 수행 상태를 포함하는 중앙 처리 장치의 상태를 나타내는 레지스터
      • 명령의 실행 중에 어떤 상황(부호 체크, 오버플로 여부, 캐리 발생 여부 등) 발생에 대한 사항들을 각각 1비트로 세트시키는 레지스터를 의미
      • 프로그램 상태 워드(PSW)는 컴퓨터 시스템 내부에서 순간순간의 시스템 상태를 기록하고 있는 특별한 워드로서 이 레지스터에 기억되어 있다
    • 메이저 상태 레지스터(MSR, Major State Register)
      • CPU가 어떤 수행을 하고 있는지를 나타내는 레지스터
